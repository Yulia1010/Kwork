Нейросетевой комплекс для распознавания
текстов и изображений во входящих письмах

Программный комплекс предназначен для обработки входных текстов, преобразования их в векторную форму по технологии Word2Vec и определения тематики.
Для распознавания входящих текстов, используется созданная и обученная нейронная сеть LSTM. Кроме того, для распознавания изображений (логотипов) характерных сервисов используется созданная и обученная нейронная сеть CNN. Основная программа явлется агрегатором комплекса программ и модулей, предназначенных для формирования и обучения векторного словаря (W2V.exe), обучения (дообучения) нейросети FCNN (Neuro.exe), обучения (дообучения) нейросети CNN (LinImg.exe), модуля лигвистического анализа и построения списка опорных слов и модуля связи с удаленными серверами HTTP и FTP (Synchro.exe). Программное обеспечение создано в среде Lazarus - открытой среде разработки программного обеспечения на языке Object Pascal для компилятора Free Pascal Compiler.
Графический интерфейс программного комплекса (GUI), выполненный в стиле Luna, с использованием оригинальной цветовой палитры и визуальными эффектами, разрабатывался как простой, удобный и интуитивно понятный – т.е. максимально дружественный (usability).
Для устойчивой и безопасной работы, в программе предусмотрена возможность работы в двух режимах: в режиме оператора – программа полностью работоспособна, но без возможности изменения настроек, и в режиме администратора - программа полностью работоспособна, с возможности изменения любых настроек.
Просмотр и определение категорий осуществляется через меню программы «Сервисы». При этом открывается окно, где можно просмотреть список существующих категорий отображенный в виде дерева.
